# 🌍 Manikanta Travels — Website

A responsive travel website built using **HTML, CSS, and Bootstrap 5**.  
It showcases travel services, vehicles, contact details, and user authentication pages.

---

## 🚀 Features

- 🏠 Home page with image carousel  
- 🚌 Travel section (vehicle options with accordion)  
- ℹ️ About section with company info  
- 📞 Contact page with:
  - Email, phone, location  
  - Google Maps integration  
  - Contact form  
- 🔐 Login page  
- 📝 Signup page  
- 📱 Fully responsive using Bootstrap 5  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3 (`travel.css`)  
- Bootstrap 5  
- SVG Icons  

---

---

## ▶️ How to Run

1. Download or clone the project  
2. Open `index.html` in your browser
